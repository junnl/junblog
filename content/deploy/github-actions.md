GitHub Actions 是 GitHub 提供的持续集成和持续部署（CI/CD）服务，可以自动化你的软件开发工作流程。

## 什么是 GitHub Actions？

GitHub Actions 让你可以在 GitHub 仓库中直接创建自定义的软件开发生命周期工作流程。

## 基本概念

- **Workflow（工作流）**: 自动化过程的配置文件
- **Job（作业）**: 工作流中的一组步骤
- **Step（步骤）**: 作业中的单个任务
- **Action（动作）**: 可重用的命令单元
