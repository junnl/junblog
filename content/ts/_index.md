# TypeScript 从入门到实战

> 专为有 JavaScript 基础的后端开发者设计的 TypeScript 学习路径

## 为什么要学 TypeScript？

作为一个有经验的程序员，你一定深知**类型系统**的重要性。TypeScript 正是为 JavaScript 带来了你熟悉的类型安全：

- 🔒 **编译时类型检查** - 像 Java/C# 一样在编译期发现错误
- 🧠 **智能代码提示** - IDE 自动补全、重构支持
- 📖 **代码即文档** - 类型定义就是最好的 API 文档
- 🚀 **渐进式采用** - 可以逐步迁移现有 JS 项目

## 学习建议

1. **边学边练** - 每个章节都有代码示例，建议亲手敲一遍
2. **对比 JS** - 文中会对比 JS 写法，帮助你理解差异
3. **联系后端** - 会类比 Java/C# 等语言的概念
4. **实战优先** - 第 9 章会结合 Vue/React 讲解实际应用

## 前置要求

- ✅ 熟悉 JavaScript（ES6+）
- ✅ 了解 Vue 或 React 基础
- ✅ 有任意后端语言经验（Java/C#/Go/Python 等）
- ✅ 安装 Node.js 18+
